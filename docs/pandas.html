1. M√≥dulo 1: Introducci√≥n a Pandas

1.1. ¬øQu√© es Pandas?

Pandas es una librer√≠a de Python especializada en el an√°lisis y manipulaci√≥n de datos. Facilita el trabajo con datos estructurados (tabulares) mediante estructuras llamadas DataFrames y Series. Es ampliamente utilizada en la ciencia de datos y el an√°lisis financiero, entre otros campos.

1.2. Instalaci√≥n de Pandas

Para instalar Pandas, utilizamos el gestor de paquetes pip:

# Instalaci√≥n
pip install pandas

Una vez instalada, se puede importar la librer√≠a en cualquier script o entorno de Python:

# Importaci√≥n de la librer√≠a
import pandas as pd

1.3. Lectura de Datos de un Archivo CSV con Pandas

La librer√≠a Pandas proporciona una forma sencilla y eficiente de leer y manipular datos en formato CSV. A continuaci√≥n, se describen los pasos para realizar la lectura de datos desde un archivo CSV y algunas operaciones b√°sicas que puedes llevar a cabo con el DataFrame resultante.

1.3.1. Leer un Archivo CSV

Puedes leer un archivo CSV utilizando la funci√≥n pd.read_csv(). A continuaci√≥n se muestra un ejemplo b√°sico.

# Leer un archivo CSV
df = pd.read_csv('ruta/al/archivo.csv')

# Mostrar las primeras 5 filas del DataFrame
print(df.head())

1.3.2. Par√°metros Comunes de read_csv()

La funci√≥n read_csv() tiene varios par√°metros √∫tiles que puedes ajustar seg√∫n tus necesidades:

  • sep: Especifica el delimitador. Por defecto es ,, pero puede cambiarse a ;, \t, etc.

  • header: Indica si la primera fila debe ser utilizada como cabecera. Puedes establecerlo en None si no tienes cabecera.

  • index_col: Define qu√© columna se usar√° como √≠ndice del DataFrame.

  • usecols: Permite especificar qu√© columnas leer.

  • dtype: Especifica el tipo de datos a usar para las columnas.

Ejemplo con par√°metros:

# Leer un archivo CSV con delimitador punto y coma y establecer la primera columna como √≠ndice
df = pd.read_csv('ruta/al/archivo.csv', sep=';', index_col=0)

# Mostrar las primeras 5 filas del DataFrame
print(df.head())

1.3.3. Manejo de Errores Comunes

Al leer un archivo CSV, es posible que te enfrentes a ciertos errores comunes, como archivos faltantes, datos mal formateados, o problemas de codificaci√≥n. Aseg√∫rate de manejar estas situaciones de manera adecuada:

  • FileNotFoundError: Aseg√∫rate de que la ruta del archivo es correcta.

  • UnicodeDecodeError: Prueba diferentes codificaciones, como encoding='utf-8' o encoding='latin1'.

Ejemplo de manejo de errores:

try:
    df = pd.read_csv('ruta/al/archivo.csv')
except FileNotFoundError:
    print("El archivo no fue encontrado.")
except UnicodeDecodeError:
    print("Error en la codificaci√≥n del archivo.")

1.3.4. Inspecci√≥n de Datos

Una vez que hayas le√≠do los datos en un DataFrame, puedes utilizar varias funciones de Pandas para inspeccionar y manipular los datos:

  • df.info(): Muestra informaci√≥n resumida sobre el DataFrame.

  • df.describe(): Genera estad√≠sticas descriptivas de las columnas num√©ricas.

  • df.columns: Devuelve el nombre de las columnas del DataFrame.

Ejemplo de inspecci√≥n:

# Inspeccionar el DataFrame
print(df.info())
print(df.describe())
print(df.columns)

1.3.5. Escritura de Datos en un Archivo CSV

Adem√°s de leer archivos CSV, Pandas tambi√©n permite escribir DataFrames en archivos CSV utilizando la funci√≥n df.to_csv(). Puedes especificar la ruta y algunos par√°metros opcionales.

Ejemplo de escritura:

# Escribir el DataFrame en un archivo CSV
df.to_csv('ruta/al/nuevo_archivo.csv', index=False)

# Opciones adicionales
# df.to_csv('ruta/al/nuevo_archivo.csv', sep=';', encoding='utf-8', columns=['columna1', 'columna2'])

1.4. Creaci√≥n de DataFrames y Series

Los DataFrames y Series son las estructuras fundamentales en Pandas.

  • Una Serie es una estructura unidimensional similar a un array o lista.

  • Un DataFrame es una estructura bidimensional que se asemeja a una tabla, con etiquetas para las filas y columnas.

1.4.1. Creaci√≥n de una Serie

import pandas as pd

# Crear una Serie desde una lista
serie = pd.Series([10, 20, 30, 40])
print(serie)

1.4.2. Creaci√≥n de un DataFrame

# Crear un DataFrame desde un diccionario
datos = {'Nombre': ['Ana', 'Luis', 'Mar√≠a'], 'Edad': [23, 25, 22]}
df = pd.DataFrame(datos)
print(df)

2. M√≥dulo 2: Operaciones b√°sicas con DataFrames

2.1. Indexaci√≥n y selecci√≥n de datos

Pandas ofrece varias formas de acceder y seleccionar datos dentro de un DataFrame. Las m√°s comunes son:

  • Acceso a columnas: Puedes seleccionar columnas utilizando su nombre.

  • .loc[]: Permite seleccionar filas y columnas por etiquetas.

  • .iloc[]: Permite seleccionar filas y columnas por √≠ndices.

import pandas as pd

# Crear un DataFrame
df = pd.DataFrame({'Nombre': ['Ana', 'Luis', 'Mar√≠a'], 'Edad': [23, 25, 22]})

# Seleccionar una columna
print(df['Nombre'])

# Seleccionar filas con loc
print(df.loc[0])  # Selecciona la primera fila

# Seleccionar por √≠ndice con iloc
print(df.iloc[1])  # Selecciona la segunda fila

2.2. Filtrado de datos

El filtrado de datos consiste en aplicar condiciones para obtener subconjuntos espec√≠ficos de un DataFrame. Las condiciones se expresan como comparaciones que resultan en valores booleanos.

# Filtrar filas donde la edad es mayor que 23
df_filtrado = df[df['Edad'] > 23]
print(df_filtrado)

Tambi√©n es posible combinar m√∫ltiples condiciones con operadores l√≥gicos como & (AND) y | (OR).

# Filtrar filas donde la edad es mayor que 23 y el nombre es 'Luis'
df_filtrado = df[(df['Edad'] > 23) & (df['Nombre'] == 'Luis')]
print(df_filtrado)

2.3. Modificaci√≥n de datos

Modificar datos en un DataFrame incluye agregar, modificar o eliminar columnas y filas.

2.3.1. Agregar columnas

# Agregar una nueva columna al DataFrame
df['Ciudad'] = ['Madrid', 'Barcelona', 'Sevilla']
print(df)

2.3.2. Modificar valores de una columna

# Modificar los valores de una columna espec√≠fica
df['Edad'] = df['Edad'] + 1  # Aumentar la edad en 1
print(df)

2.3.3. Eliminar columnas y filas

# Eliminar una columna
df = df.drop('Ciudad', axis=1)

# Eliminar una fila por √≠ndice
df = df.drop(1)  # Eliminar la fila con √≠ndice 1
print(df)

3. M√≥dulo 3: Limpieza de datos

3.1. Manejo de valores faltantes (NaN)

En los conjuntos de datos reales, es com√∫n encontrar valores faltantes representados como NaN (Not a Number). Pandas ofrece varias funciones para identificar, eliminar o reemplazar estos valores.

3.1.1. Identificaci√≥n de valores faltantes

import pandas as pd

# Crear un DataFrame con valores faltantes
df = pd.DataFrame({'Nombre': ['Ana', 'Luis', 'Mar√≠a'], 'Edad': [23, None, 22], 'Ciudad': [None, 'Barcelona', 'Sevilla']})

# Detectar valores faltantes
print(df.isna())

3.1.2. Eliminar valores faltantes

Puedes eliminar las filas o columnas que contengan valores faltantes utilizando dropna().

# Eliminar filas con valores faltantes
df_sin_nan = df.dropna()
print(df_sin_nan)

# Eliminar columnas con valores faltantes
df_sin_nan_col = df.dropna(axis=1)
print(df_sin_nan_col)

3.1.3. Rellenar valores faltantes

Otra opci√≥n es reemplazar los valores NaN con valores espec√≠ficos usando fillna().

# Rellenar los valores faltantes con un valor espec√≠fico
df_filled = df.fillna({'Edad': 0, 'Ciudad': 'Desconocido'})
print(df_filled)

3.2. Eliminaci√≥n de duplicados

Los datos duplicados pueden afectar los an√°lisis y los resultados. Pandas permite identificar y eliminar estas duplicaciones con drop_duplicates().

# Crear un DataFrame con duplicados
df_dup = pd.DataFrame({'Nombre': ['Ana', 'Luis', 'Ana'], 'Edad': [23, 25, 23]})

# Eliminar filas duplicadas
df_sin_duplicados = df_dup.drop_duplicates()
print(df_sin_duplicados)

3.3. Conversi√≥n de tipos de datos

A veces es necesario convertir el tipo de datos de una columna, especialmente cuando se trabajan con fechas o valores num√©ricos que se han importado como cadenas de texto.

3.3.1. Convertir tipos de datos

Usa astype() para cambiar el tipo de datos de una columna.

# Crear un DataFrame con tipos incorrectos
df_tipos = pd.DataFrame({'Nombre': ['Ana', 'Luis'], 'Edad': ['23', '25']})

# Convertir la columna 'Edad' a tipo entero
df_tipos['Edad'] = df_tipos['Edad'].astype(int)
print(df_tipos.dtypes)

3.3.2. Manejo de fechas

Para trabajar con fechas, Pandas proporciona la funci√≥n to_datetime() que convierte cadenas de texto a objetos de fecha.

# Convertir una columna de texto a fecha
df_fechas = pd.DataFrame({'Fecha': ['2023-01-01', '2023-01-02']})
df_fechas['Fecha'] = pd.to_datetime(df_fechas['Fecha'])
print(df_fechas.dtypes)

4. M√≥dulo 4: Operaciones avanzadas

4.1. Agrupaci√≥n de datos con groupby()

La funci√≥n groupby() en Pandas permite agrupar datos en funci√≥n de los valores de una o m√°s columnas y luego aplicar funciones agregadas como sum(), mean(), count(), entre otras.

import pandas as pd

# Crear un DataFrame de ejemplo
data = {'Nombre': ['Ana', 'Luis', 'Mar√≠a', 'Ana', 'Luis'], 'Edad': [23, 25, 22, 23, 30], 'Ciudad': ['Madrid', 'Barcelona', 'Sevilla', 'Madrid', 'Barcelona']}

df = pd.DataFrame(data)

# Agrupar por la columna 'Nombre' y calcular la edad media
grupo = df.groupby('Nombre')['Edad'].mean()
print(grupo)

Tambi√©n es posible agrupar por varias columnas y aplicar varias funciones agregadas.

# Agrupar por 'Ciudad' y 'Nombre', y calcular la suma y la cuenta de las edades
grupo = df.groupby(['Ciudad', 'Nombre']).agg({'Edad': ['sum', 'count']})
print(grupo)

4.2. Ordenaci√≥n de datos

Pandas permite ordenar los datos en un DataFrame utilizando el m√©todo sort_values(). Puedes ordenar por una o m√°s columnas, y definir si quieres que el orden sea ascendente o descendente.

# Ordenar por la columna 'Edad' en orden ascendente
df_ordenado = df.sort_values('Edad')
print(df_ordenado)

# Ordenar por la columna 'Edad' en orden descendente
df_ordenado_desc = df.sort_values('Edad', ascending=False)
print(df_ordenado_desc)

Tambi√©n es posible ordenar por m√∫ltiples columnas.

# Ordenar primero por 'Ciudad' y luego por 'Edad' en cada ciudad
df_multi_orden = df.sort_values(['Ciudad', 'Edad'])
print(df_multi_orden)

4.3. Fusi√≥n y combinaci√≥n de DataFrames

Pandas ofrece varias maneras de combinar DataFrames, incluyendo las funciones merge(), concat(), y join(). Estas se utilizan para unir diferentes conjuntos de datos en funci√≥n de una clave com√∫n.

4.3.1. Fusi√≥n de DataFrames con merge()

La funci√≥n merge() se utiliza para realizar uniones tipo SQL, uniendo DataFrames en funci√≥n de una columna com√∫n.

# Crear dos DataFrames
df1 = pd.DataFrame({'ID': [1, 2, 3], 'Nombre': ['Ana', 'Luis', 'Mar√≠a']})
df2 = pd.DataFrame({'ID': [1, 2, 4], 'Ciudad': ['Madrid', 'Barcelona', 'Valencia']})

# Realizar una uni√≥n por la columna 'ID'
df_merged = pd.merge(df1, df2, on='ID', how='inner')
print(df_merged)

Los tipos de uniones disponibles son:

  • inner: Devuelve las filas que tienen coincidencia en ambas tablas.

  • left: Devuelve todas las filas de la tabla izquierda, y las coincidencias de la derecha (rellena con NaN si no hay coincidencia).

  • right: Devuelve todas las filas de la tabla derecha y las coincidencias de la izquierda.

  • outer: Devuelve todas las filas de ambas tablas, con NaN donde no haya coincidencias.

4.3.2. Concatenaci√≥n de DataFrames con concat()

La funci√≥n concat() se utiliza para concatenar DataFrames a lo largo de un eje (filas o columnas).

# Concatenar DataFrames por filas (uno debajo del otro)
df_concat = pd.concat([df1, df2], axis=0, ignore_index=True)
print(df_concat)

# Concatenar DataFrames por columnas (lado a lado)
df_concat_cols = pd.concat([df1, df2], axis=1)
print(df_concat_cols)

4.3.3. Uni√≥n de DataFrames con join()

El m√©todo join() se utiliza para unir DataFrames utilizando los √≠ndices.

# Crear dos DataFrames con √≠ndices
df1 = pd.DataFrame({'Nombre': ['Ana', 'Luis', 'Mar√≠a']}, index=[1, 2, 3])
df2 = pd.DataFrame({'Ciudad': ['Madrid', 'Barcelona', 'Sevilla']}, index=[1, 2, 3])

# Unir los DataFrames usando el √≠ndice
df_join = df1.join(df2)
print(df_join)

5. M√≥dulo 5: Visualizaci√≥n de datos

5.1. Integraci√≥n de Pandas con Matplotlib

Pandas se integra f√°cilmente con Matplotlib, una de las bibliotecas m√°s populares para la visualizaci√≥n de datos en Python. Puedes crear gr√°ficos directamente desde un DataFrame o una Serie.

5.1.1. Gr√°ficos b√°sicos con Pandas

Pandas permite crear gr√°ficos sencillos con un solo comando. Aqu√≠ hay algunos ejemplos de diferentes tipos de gr√°ficos.

import pandas as pd
import matplotlib.pyplot as plt

# Crear un DataFrame de ejemplo
df = pd.DataFrame({'Nombre': ['Ana', 'Luis', 'Mar√≠a'], 'Edad': [23, 25, 22]})

# Gr√°fico de barras
df.plot(kind='bar', x='Nombre', y='Edad')

plt.show()

5.1.2. Gr√°ficos de l√≠neas

Puedes crear gr√°ficos de l√≠neas para mostrar tendencias a lo largo del tiempo.

# Crear un DataFrame de ejemplo
df_lineas = pd.DataFrame({'Mes': ['Enero', 'Febrero', 'Marzo'], 'Ventas': [200, 300, 250]})

# Gr√°fico de l√≠neas
df_lineas.plot(kind='line', x='Mes', y='Ventas')
plt.title('Ventas por Mes')
plt.ylabel('Ventas')
plt.show()

5.2. Generaci√≥n de gr√°ficos con Seaborn

Seaborn es una biblioteca de visualizaci√≥n de datos basada en Matplotlib que proporciona una interfaz m√°s sencilla y atractiva para crear gr√°ficos estad√≠sticos.

5.2.1. Gr√°ficos de dispersi√≥n

Los gr√°ficos de dispersi√≥n son √∫tiles para visualizar la relaci√≥n entre dos variables.

import seaborn as sns

# Crear un DataFrame de ejemplo
df_seaborn = pd.DataFrame({'Edad': [23, 25, 22, 30, 29], 'Salario': [40000, 50000, 45000, 60000, 55000]})

# Gr√°fico de dispersi√≥n
sns.scatterplot(data=df_seaborn, x='Edad', y='Salario')
plt.title('Salario seg√∫n Edad')
plt.show()

5.2.2. Gr√°ficos de cajas (Boxplots)

Los boxplots son √∫tiles para mostrar la distribuci√≥n de los datos y detectar outliers.

# Gr√°fico de cajas
sns.boxplot(data=df_seaborn, x='Edad', y='Salario')
plt.title('Distribuci√≥n de Salarios seg√∫n Edad')
plt.show()

5.2.3. Gr√°ficos de barras

Puedes crear gr√°ficos de barras para comparar categor√≠as.

# Gr√°fico de barras
sns.barplot(data=df_seaborn, x='Edad', y='Salario')
plt.title('Salario promedio por Edad')
plt.show()

5.3. Personalizaci√≥n de gr√°ficos

Tanto Matplotlib como Seaborn permiten personalizar los gr√°ficos con t√≠tulos, etiquetas, colores y estilos para mejorar la presentaci√≥n visual.

# Personalizaci√≥n de un gr√°fico
plt.figure(figsize=(10, 5))
sns.barplot(data=df_seaborn, x='Edad', y='Salario', palette='viridis')
plt.title('Salario Promedio por Edad')
plt.xlabel('Edad')
plt.ylabel('Salario')
plt.show()

6. M√≥dulo 6: Importaci√≥n y exportaci√≥n de datos

6.1. Lectura de archivos CSV

Pandas facilita la lectura de archivos CSV utilizando la funci√≥n read_csv(). Esta funci√≥n permite cargar datos en un DataFrame de manera r√°pida y sencilla.

6.1.1. Lectura de un archivo CSV

Para leer un archivo CSV, simplemente especifica la ruta del archivo. Aqu√≠ hay un ejemplo:

import pandas as pd

# Leer un archivo CSV
df_csv = pd.read_csv('ruta/al/archivo.csv')
print(df_csv.head())  # Muestra las primeras filas del DataFrame

6.1.2. Opciones de lectura

Puedes personalizar la lectura de archivos CSV con varios par√°metros, como sep, header, index_col, entre otros.

# Leer un archivo CSV con un separador diferente y sin encabezado
df_csv_custom = pd.read_csv('ruta/al/archivo.csv', sep=';', header=None)
print(df_csv_custom.head())

6.2. Lectura de archivos Excel

Pandas tambi√©n permite la lectura de archivos Excel mediante la funci√≥n read_excel(). Necesitar√°s tener instalado openpyxl o xlrd para manejar archivos Excel.

6.2.1. Lectura de un archivo Excel

Para leer un archivo Excel, especifica la ruta del archivo y la hoja que deseas cargar.

# Leer un archivo Excel
df_excel = pd.read_excel('ruta/al/archivo.xlsx', sheet_name='Hoja1')
print(df_excel.head())

6.2.2. Opciones de lectura

Puedes personalizar la lectura de archivos Excel con par√°metros como header, index_col, usecols, etc.

# Leer un archivo Excel especificando columnas
df_excel_custom = pd.read_excel('ruta/al/archivo.xlsx', usecols=['A', 'B'])
print(df_excel_custom.head())

6.3. Exportaci√≥n de datos a CSV y Excel

Pandas permite exportar DataFrames a archivos CSV y Excel usando las funciones to_csv() y to_excel().

6.3.1. Exportaci√≥n a un archivo CSV

Para exportar un DataFrame a CSV, utiliza to_csv(). Puedes personalizar el formato de salida con varios par√°metros.

# Exportar un DataFrame a CSV
df_csv_export = df_csv.to_csv('ruta/al/nuevo_archivo.csv', index=False)
print("Datos exportados a CSV.")

6.3.2. Exportaci√≥n a un archivo Excel

Para exportar un DataFrame a Excel, utiliza to_excel(). Especifica el nombre del archivo y la hoja.

# Exportar un DataFrame a Excel
df_excel_export = df_excel.to_excel('ruta/al/nuevo_archivo.xlsx', sheet_name='Hoja1', index=False)

7. M√≥dulo 7: Trabajo con series temporales

7.1. Fechas en Pandas

Pandas proporciona potentes herramientas para trabajar con fechas y horas. Las fechas se pueden manejar usando el tipo de datos datetime, que permite realizar operaciones y an√°lisis temporales.

7.1.1. Creaci√≥n de objetos de fecha

Puedes crear objetos de fecha directamente desde cadenas de texto o utilizando la funci√≥n pd.to_datetime().

import pandas as pd

# Crear un DataFrame con fechas
fechas = pd.date_range(start='2023-01-01', periods=5, freq='D')
df_fechas = pd.DataFrame({'Fecha': fechas, 'Valores': [10, 20, 15, 30, 25]})
print(df_fechas)

7.1.2. Indexaci√≥n por fechas

Pandas permite indexar y seleccionar datos utilizando fechas. Esto es especialmente √∫til en an√°lisis de series temporales.

# Establecer la columna 'Fecha' como √≠ndice
df_fechas.set_index('Fecha', inplace=True)

# Seleccionar datos por fecha
print(df_fechas.loc['2023-01-03'])

7.2. Resampleo de datos

El resampleo es una t√©cnica que permite reorganizar y resumir datos temporales a diferentes frecuencias. Esto es √∫til para obtener promedios, sumas, conteos, etc., en diferentes intervalos de tiempo.

7.2.1. Resampleo de datos

La funci√≥n resample() permite especificar la frecuencia deseada y aplicar funciones de agregaci√≥n.

# Crear un DataFrame con datos aleatorios y fechas
data = {'Fecha': pd.date_range(start='2023-01-01', periods=10, freq='D'), 'Valores': [10, 20, 15, 30, 25, 10, 5, 20, 15, 10]}
df_resample = pd.DataFrame(data).set_index('Fecha')

# Resamplear datos a frecuencia semanal y calcular la suma
resample_semanal = df_resample.resample('W').sum()
print(resample_semanal)

7.2.2. Otras funciones de resampleo

Adem√°s de sum(), se pueden aplicar otras funciones como mean(), max(), min(), entre otras.

# Resamplear y calcular la media semanal
resample_media_semanal = df_resample.resample('W').mean()
print(resample_media_semanal)

7.2.3. Ejemplo de resampleo con agregaci√≥n m√∫ltiple

Se pueden aplicar m√∫ltiples funciones de agregaci√≥n al mismo tiempo utilizando agg().

# Resamplear y aplicar m√∫ltiples funciones de agregaci√≥n
resample_multifunciones = df_resample.resample('W').agg({'Valores': ['sum', 'mean', 'max']})
print(resample_multifunciones)
